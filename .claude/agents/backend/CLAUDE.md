## 백엔드 에이전트 규칙

이 디렉토리의 에이전트가 코드를 작성할 때 위 코딩 규칙을 해당 언어에 맞춰 따른다.

- Rust 에이전트(`rust-backend-architect`, `rust-backend-developer`, `build-error-resolver`) → Rust 규칙 적용
- Java 에이전트(`java-backend-architect`, `java-backend-developer`) → Java 규칙 적용 (레거시 SB 2.5 + 모던 SB 3.x 모두 커버)
