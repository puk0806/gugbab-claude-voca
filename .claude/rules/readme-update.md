# README 업데이트 규칙

에이전트·스킬·훅을 추가/수정/삭제/이름변경 시 README.md도 반드시 함께 업데이트한다.

## 변경 유형별 체크리스트

| 변경 유형 | README 반영 항목 |
|----------|----------------|
| **추가** | 목록 섹션 항목 추가, 스킬 수 증가, 업데이트 로그 기록 |
| **삭제** | 목록 섹션 항목 제거, 스킬 수 감소, 구조도 파일명 제거 |
| **이름변경** | 구조도·목록 양쪽 모두 새 이름으로 수정 |
| **이동** | 구조도 경로 업데이트, 목록 링크 경로 수정 |

## 업데이트 로그 규칙

- 날짜별 단일 행 유지
- 같은 날 여러 변경은 기존 행에 합산 (새 행 추가 금지)
- 형식: `| YYYY-MM-DD | 변경 내용 요약 |`
