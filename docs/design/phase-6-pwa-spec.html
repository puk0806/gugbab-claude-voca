03Service Worker

vite-plugin-pwa(Workbox 기반) · autoUpdate 전략. 1인용이라 prompt UI 불필요.

항목결정근거
도구vite-plugin-pwa사실상 표준 (Vite 공식 권장)
갱신 전략registerType: 'autoUpdate'1인용. PRD #6(콘텐츠 갱신 UI)은 P2로
skipWaitingtrue새 SW 즉시 활성
clientsClaimtrue모든 탭 즉시 제어
precache빌드 산출물(JS·CSS·HTML) + public/data/**전체 오프라인
runtime cache없음외부 의존 0 (system-ui · 백엔드 없음)
SW 라이프사이클 (사용자 시점):
  1. 첫 방문 → install: 빌드 산출물 + 콘텐츠 JSON 모두 precache (백그라운드)
  2. 두 번째 방문부터 → cache-first: 즉시 렌더 (네트워크 X)
  3. 새 배포 감지 → 새 SW 백그라운드 다운로드 → 다음 페이지 진입 시 자동 적용
  4. 네트워크 끊김 → 오프라인 동작 그대로 (모든 화면 사용 가능)
