# research-reviewer

> 리서치 보고서 초안을 5항목 품질 기준으로 평가해 PASS 또는 GAPS를 반환하는 리뷰 전담 에이전트

| 항목 | 내용 |
|------|------|
| 파일 | `.claude/agents/research/research-reviewer.md` |
| 모델 | Opus |
| 도구 | Read |
| 호출 | `deep-researcher`가 자동 호출 (서브에이전트) |

## 역할

`deep-researcher`가 초안 작성 후 자동으로 호출합니다. 5개 품질 기준으로 평가하고 부족한 항목이 있으면 추가 검색 제안과 함께 GAPS를 반환합니다.

## 평가 기준 5항목

| 기준 | 통과 조건 |
|------|-----------|
| 커버리지 | 3축(논문/오픈소스/기업사례) 각각 최소 3개 소스 |
| 최신성 | 2024-2026 자료가 1개 이상 포함 |
| 구체성 | 수치/성과/코드 예시가 전체 항목의 50% 이상 |
| 관련성 | 모든 항목이 원본 주제와 직접 연관 |
| 깊이 | 시사점 필드가 표면적 요약이 아닌 구체적 인사이트 |

## 반환 형식

```
# PASS인 경우
상태: PASS
통과: 5/5

# GAPS인 경우
상태: GAPS
통과: X/5
미통과 항목:
  - {기준명}: {구체적 문제}
    추가 검색 제안: "축={축명}, 쿼리=[...]"
```

## 직접 호출 예시

이미 작성된 리서치 문서의 품질을 검증할 때 직접 호출합니다.

```
파일: docs/research/my-research.md
원본 주제: LLM 기반 코드 리뷰 자동화
```
