# agent-creator

> 개발 관련 서브에이전트 MD 파일을 대화형으로 설계하고 생성하는 에이전트

| 항목 | 내용 |
|------|------|
| 파일 | `.claude/agents/meta/agent-creator.md` |
| 모델 | Opus |
| 도구 | Read, Write, Glob |
| 호출 | 자동 또는 `@agent-creator` |

## 역할

새로운 에이전트가 필요할 때 직접 MD 파일을 작성하는 대신, 원하는 기능을 설명하면 요구사항을 파악하고 `.claude/agents/`에 바로 저장 가능한 완성된 MD 파일을 생성합니다.

## 동작 흐름

1. 요구사항 수집 (역할, 입력/출력 형태, 사용 빈도, 오케스트레이터 여부)
2. 모델·도구 선택 (CLAUDE.md 설계 원칙 기준)
3. MD 파일 생성 + 품질 체크리스트 검토
4. `.claude/agents/{category}/{name}.md` 저장

## 호출 예시

```
코드 리뷰 에이전트 만들어줘
PR 요약 자동화하는 에이전트 필요해
API 문서 자동 생성 에이전트 만들어줘
```

## 주의사항

- 생성된 에이전트는 실제 사용 전 테스트 권장
- 테스트 전 main 브랜치 직접 커밋 금지 (CLAUDE.md 규칙)
